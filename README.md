# macOS dotfiles

My configuration files for the various programs I use on my macOS machine.

## Installation

```bash
bash -c "$(curl -fsSL \
  https://raw.githubusercontent.com/noib3/dotfiles/macOS/bootstrap/bootstrap.sh)"
```
